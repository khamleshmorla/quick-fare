Privacy Policy

Information Collection and Use

QuickFare collects and uses location data to provide fare estimates for ride-sharing services. We do not store your location data permanently and only use it to fetch real-time fare estimates.

Uber Integration

When using the Uber fare estimation feature, we use Uber's API to fetch real-time fare estimates. We do not store any personal information or ride history.

Data Security

We implement appropriate security measures to protect your data. All API communications are encrypted using HTTPS.
